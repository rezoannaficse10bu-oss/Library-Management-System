Introduction

This project is a PHP-based dynamic Library Management System developed using Object-Oriented Programming (OOP) and MySQL. It allows administrators to manage books, students, and borrowing records efficiently. The system provides features to add, update, delete, issue, and return books, along with maintaining proper records. This project demonstrates clean OOP structure, secure database connectivity, and organized data management.

Quick Start Running the Project

git clone https://github.com/rezoannaficse10bu-oss/library-management-system.git

Download and install XAMPP from https://www.apachefriends.org/download.html

Copy the cloned project folder into the htdocs directory inside the XAMPP installation folder

Start Apache and MySQL from the XAMPP Control Panel

Open phpMyAdmin and import the provided database file (e.g., library_db.sql)

Visit http://127.0.0.1/library-management-system in your browser to access the website
